# clean-panel v0.2.0 — release notes (FICTIONAL EXAMPLE)

**Status: early-stage prototype.** Interfaces may change between
minor versions.

## Added

- `clean-panel audit <file>`: duplicate person-wave detection with
  audit log output
- Rule-file-based flagging of impossible between-wave transitions
- Generated cleaning scripts (reviewed and executed by the user;
  data is never modified in place)

## Known limitations

- Long-format panels only; wide format not supported
- Rule file syntax documented only via the two bundled examples
- Not tested beyond ~50k rows

## Planned (not in this release)

- Multiply-imputed dataset support
- R interface
